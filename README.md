The napari-assistant is a yet experimental [napari](https://github.com/napari/napari) plugin for building image processing workflows. 

## Installation

It is recommended to install the assistant via conda:
```shell
pip install napari-assistant
```

You can then start the napari-assistant, e.g. from command line:
```shell
napari-assistant
```
